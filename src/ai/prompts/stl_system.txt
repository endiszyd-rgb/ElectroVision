## 1. MATERIAŁY DO DRUKU 3D
════════════════════════════════════════════════════════

### PLA (Polylactic Acid)
- Temperatura dyszy: 180-230°C, stół: 20-60°C (bez ogrzewania)
- Tg: 55-60°C — NIE do zastosowań w samochodzie, na słońcu
- Gęstość: 1.24 g/cm³
- Zastosowanie: prototypy wewnętrzne, makiety, demonstratory
- Zalety: tani (20-25 zł/kg), łatwy w druku, biodegradowalny
- Wady: kruchy, hygroskopijny, słaba odporność UV i chemiczna

### PETG — ZALECANY dla elektroniki
- Temperatura dyszy: 230-250°C, stół: 70-90°C
- Tg: 70-80°C
- Gęstość: 1.27 g/cm³
- Zalety: twardy + elastyczny, chemoodporny (IPA, aceton słabo), mało wilgoci
- Wady: stringing (retraction 1-2mm Direct Drive), trudniejszy niż PLA
- Parametry Cura: 245°C/85°C, prędkość 50mm/s, chłodzenie 40-60%

### ABS — dla wymagań mechanicznych/termicznych
- Temperatura dyszy: 230-250°C, stół: 100-110°C (konieczne!)
- Tg: 100-115°C, odporny do 90°C
- WYMAGA: obudowanej drukarki, wentylacji (opary styrenu!)
- Zalety: szlifowanie, acetone smoothing, post-processing
- Zastosowanie: obudowy przemysłowe, elementy mechaniczne

### ASA — outdoor
- Jak ABS + odporność UV do 5000h
- Stosuj dla: urządzeń zewnętrznych, ogrodowych, przemysłowych

### TPU (Shore A 87-98)
- Elastyczny, antywibracyjny
- Temperatura dyszy: 210-230°C, TYLKO Direct Drive, prędkość ≤30mm/s
- Zastosowanie: uszczelki, osłony, uchwyty kabli, pady antywibracyjne

### Resin (SLA/DLP)
- Precyzja: 0.025-0.05mm vs FDM 0.1-0.2mm
- Kruchy, wymaga UV curing po wydruku
- Stosuj dla: małych, precyzyjnych elementów, złączy

## 3. PARAMETRY OBUDÓW PCB
════════════════════════════════════════════════════════

### Grubość ścianek
- FDM minimum: 1.2mm (= 3× linia 0.4mm dyszy)
- FDM zalecane: 2.0-2.5mm (solidna obudowa)
- Miejsca mocowań (standoffy): 3.0-4.0mm
- Narożniki wewnętrzne: R ≥ 0.5mm
- Zewnętrzne zaokrąglenia: R = 1.0-3.0mm (estetyka)

### Standoffy (filarki montażu PCB)
- Zewnętrzna średnica: 5.0mm (6mm dla M3)
- Otwory: M2=⌀2.2mm (luz), M3=⌀3.4mm (luz)
- Gwint samo-gwintujący: M2=⌀1.8mm, M3=⌀2.7mm
- Wysokość: 3-5mm (PCB spoczywa na standoffach)
- Pozycja: odpowiada otworom KiCad MountingHole_3.2mm_M3

### Wieko i zamknięcia
1. **Snap-fit**: lip 1.5×1.5mm + rowek, wcisk 0.3-0.5mm, kąt 30-45°
2. **Śruby M3**: 4 narożniki z heat inserts (mosiężne wkładki)
3. **Magnesy**: ⌀6×2mm neodym, wcisk -0.1mm
4. **Zatrzask boczny**: sprężyna 0.8mm, wcisk 0.4mm

### IP Rating (ochrona przed wodą/pyłem)
- IP54: rowek 1.5×1.5mm + O-ring ⌀3.0mm NBR, kompresja 20%
- IP65: rowek 2.0×2.0mm + O-ring ⌀3.5mm, dławiki PG7/M12
- IP67: precyzyjna obróbka cnc + uszczelka silikonowa wulkanizowana

## 4. REGUŁY DRUKU 3D (DFM)
════════════════════════════════════════════════════════

### Nawisy (Overhangs)
- FDM bez podpór: do 45° od pionu (≤45° od poziomego sufitu)
- PETG/ABS: do 60° możliwe przy powolnym druku i chłodzeniu
- >45°: potrzebne podpory (ustaw w slicer: Support Overhang Angle)
- Projektuj z myślą o druku BEZ podpór (oszczędność czasu + lepsza jakość)

### Mosty (Bridges)
- Bez podpór do 50-60mm (PETG lepszy niż PLA dla mostów)
- Chłodzenie 100% podczas druku mostów
- Tip: lekki szew po środku mostu poprawia jakość

### Minimalne elementy
- FDM: min grubość = 2× dyszy = 0.8mm (niestabilne)
- Zalecane minimum: 1.2mm (3 linie)
- Haki, zaczepy, małe klipsy: ≥2.0mm
- Gwint drukowany: M6+ (M3 możliwy z samo-gwintowaniem)

### Orientacja na stole
- Otworem do góry: brak podpór wewnątrz, lepsza powierzchnia
- Maksymalna wytrzymałość: warstwy prostopadle do sił
- Otwory okrągłe: drukuj pionowo dla lepszej okrągłości

## 5. CADQUERY — KLUCZOWE WZORCE
════════════════════════════════════════════════════════

```python
import cadquery as cq
from cadquery import exporters

# Parametryczna obudowa PCB
def make_enclosure(pcb_w, pcb_l, height, wall=2.0, margin=3.0, r=2.0):
    ext_w = pcb_w + 2*margin + 2*wall
    ext_l = pcb_l + 2*margin + 2*wall
    ext_h = height + wall

    body = (cq.Workplane("XY")
        .box(ext_l, ext_w, ext_h)
        .edges("|Z").fillet(r)      # narożniki pionowe
        .shell(-wall)               # wydrążenie
    )
    return body

# Dodanie standoffu
def add_standoff(body, x, y, h=4.0, or_=2.5, hole_r=1.1, z_offset=2.0):
    so = (cq.Workplane("XY")
        .center(x, y)
        .cylinder(h, or_)
        .faces(">Z").hole(hole_r*2, h)
        .translate((0, 0, z_offset))
    )
    return body.union(so)

# Wycięcie na złącze
def add_cutout(body, face_sel, cx, cy, w, h, depth=2.5):
    return (body
        .faces(face_sel).workplane()
        .center(cx, cy)
        .rect(w, h)
        .cutBlind(-(depth + 0.1))
    )

# Selektory ścian:
# ">Y"  — ściana frontowa (y+)
# "<Y"  — ściana tylna  (y-)
# ">X"  — ściana prawa
# "<X"  — ściana lewa
# ">Z"  — góra,  "<Z" — dół

# Eksport
exporters.export(body, "enclosure.stl")  # dla slicera
exporters.export(body, "enclosure.step") # dla Fusion 360
```

## 6. FUSION 360 WORKFLOW
════════════════════════════════════════════════════════

### Import STEP z ElectroVision
1. Fusion 360: File → Open → wybierz .step
2. Fusion konwertuje bryłę na parametryczną (B-rep)
3. Możliwa edycja: double-click na krawędź → zmień wymiar
4. Split Body: Model → Modify → Split Body (dla wieka osobno)
5. Eksport STL: Right click → Save As Mesh → STL

### Konwencja pliku STEP
- Format: ISO-10303-21 (AP214 lub AP242)
- Nazwy warstw: Enclosure_Body, Enclosure_Lid, Standoffs, PCB
- CadQuery generuje AP214 domyślnie — kompatybilne z Fusion 360

### Modyfikacja w Fusion po imporcie
- Press/Pull: zmiana grubości ścianki
- Fillet: dodanie zaokrąglenia
- Shell: wydrążenie modelu
- Combine: złączenie bryły z wieczkiem

## 7. PARAMETRY SLICERA
════════════════════════════════════════════════════════

### Profil PETG (Cura/PrusaSlicer)
- Layer height: 0.2mm (0.15mm = wysoka jakość)
- Wall thickness: 2.0mm (= 5 linii 0.4mm)
- Top/Bottom layers: 5 (= 1mm)
- Infill: 25-40%, wzór Gyroid lub Grid
- Temperature nozzle: 245°C
- Temperature bed: 85°C
- Print speed: 50mm/s (perimeter), 80mm/s (infill)
- Fan: 50-60% (nie 100% jak PLA)
- Retraction: 1.5mm / 30mm/s (Direct Drive), 5mm / 45mm/s (Bowden)
- Ironing: TAK dla górnych powierzchni

### Szacunek materiału
- Gęstość PETG: 1.27 g/cm³, PLA: 1.24 g/cm³
- 1m filamentu 1.75mm ≈ 3g (PETG)
- Obudowa 100×80×40mm, wall=2mm, infill=25% ≈ 40-60g ≈ 13-20m
